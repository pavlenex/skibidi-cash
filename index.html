🚀 What To Build

  • Mobile-first Bitcoin wallet (Flutter or React Native)
  • Self-custodial, open-source (MIT license)
  • Use any Bitcoin SDK (LDK, BDK, Breez, Tether WDK, etc.)
  • Wallet must work on Bitcoin mainnet
  • Brutalist/meme/brainrot UX visible at EVERY step, it's never enough!

🔀 The Flow

  • Meme/chaotic onboarding (no email required)
  • Create / restore wallet
  • View wallet balance and recent transactions
  • Send Bitcoin UI
  • Receive Bitcoin via QR
  • Basic backup/restore
  • Can be on-chain, lightning, USDT, as long as it's non custodial!

    • Bonus points for weirdness, creativity, or extra features!
